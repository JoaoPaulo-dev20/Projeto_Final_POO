# Backend Python Django
Desenvolvimento do Backend do projeto de gerenciamento de mesas.

#### Requisitos locais
- Python 3.10+ (ou compatível)
- PowerShell/terminal

Instale as dependências:

```
cd backend
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

Executar

```
cd Backend, reserveaqui
python manage.py runserver
```
